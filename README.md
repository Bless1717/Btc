## Installation
To get started with Ace Genius, follow these simple steps:

1. Clone the repository:
```bash
$ git clone https://github.com/yourusername/ace-genius.git
```

2. Install dependencies:
```bash
$ npm install
```

3. Start the application:
```bash
$ npm start
```
